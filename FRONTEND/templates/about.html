About Project


This project aims to classify realistic 12-lead ECG(12 leads is from 12 direction the heart is viewed and took ecg) signal images using advanced deep learning techniques. The primary objective is to identify various heart conditions such as arrhythmias and normal heart patterns by analyzing ECG images. The project employs a hybrid deep learning approach that integrates Convolutional Neural Networks (CNN), MobileNet, DenseNet, and an ensemble model combining MobileNet with Long Short-Term Memory (LSTM) networks. These models are trained on a dataset of labeled ECG images sourced from Kaggle, providing a robust foundation for developing an automated classification system. The goal is to create a high-accuracy model that can assist in the early detection of cardiac abnormalities, thereby enhancing the diagnostic process in cardiology.
